# 🧑‍💻 Lección: Form Binding en Vue 3 (Composition API)

------------------------------------------------------------------------

## 1️⃣ Introducción

Uno de los pilares fundamentales de Vue es su capacidad para crear
**interfaces reactivas** de forma simple y eficiente.

Dentro de este enfoque, el manejo de formularios es clave porque
permite:

-   Recolectar datos del usuario
-   Sincronizar datos automáticamente
-   Reducir código imperativo
-   Mantener el estado siempre consistente

Vue lo logra mediante el sistema de **Form Binding** usando `v-model`.

------------------------------------------------------------------------

## 2️⃣ ¿Qué es Binding?

**Binding** es el proceso mediante el cual un valor del modelo de datos
se conecta automáticamente con la vista.

Puede ser:

-   One-Way Binding → Modelo → Vista
-   Two-Way Binding → Modelo ↔ Vista

En formularios usamos **Two-Way Binding**.

------------------------------------------------------------------------

## 3️⃣ Two-Way Binding con `v-model`

`v-model` crea una vinculación bidireccional automática.

Internamente equivale a:

``` vue
v-bind:value + @input
```

-   Cuando el usuario escribe → se actualiza el modelo\
-   Cuando el modelo cambia → se actualiza el input

# Uso básico con Composition API

## 🟢 Input

``` vue
<script setup>
import { ref } from 'vue'

const mensaje = ref('')
</script>

<template>
  <input v-model="mensaje" placeholder="Escribe algo..." />
  <p>{{ mensaje }}</p>
</template>
```

# Modificadores de v-model

## `.lazy`

``` vue
<input v-model.lazy="mensaje" />
```

## `.number`

``` vue
<input type="number" v-model.number="edad" />
```

## `.trim`

``` vue
<input v-model.trim="nombre" />
```

# Ejemplo práctico: Formulario de registro

``` vue
<script setup>
import { reactive } from 'vue'

const usuario = reactive({
  nombre: '',
  email: '',
  password: '',
  aceptaTerminos: false
})

function registrar() {
  console.log(usuario)
}
</script>

<template>
  <form @submit.prevent="registrar">
    <input v-model="usuario.nombre" />
    <input v-model="usuario.email" type="email" />
    <input v-model="usuario.password" type="password" />
    <input v-model="usuario.aceptaTerminos" type="checkbox" />
    <button type="submit">Registrar</button>
  </form>
</template>
```

# Validación con Composition API

``` vue
<script setup>
import { ref, computed } from 'vue'

const email = ref('')
const password = ref('')

const formularioValido = computed(() =>
  email.value.includes('@') &&
  password.value.length >= 6
)
</script>
```

# Caso avanzado: Configurador de Producto

``` vue
<script setup>
import { reactive, computed } from 'vue'

const configuracion = reactive({
  modelo: 'basico',
  extras: []
})

const precios = {
  modelos: {
    basico: 800,
    avanzado: 1200
  },
  extras: {
    'Pantalla táctil': 200
  }
}

const precioTotal = computed(() => {
  let total = precios.modelos[configuracion.modelo]

  configuracion.extras.forEach(extra => {
    total += precios.extras[extra]
  })

  return total
})
</script>
```
