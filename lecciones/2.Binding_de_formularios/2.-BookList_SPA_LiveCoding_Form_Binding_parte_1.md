# 📚 BookList SPA --- Gestor de Libros Interactivo

## 🎯 Objetivo de la Clase

Construir el formulario reactivo de libros aplicando:

-   Composition API
-   v-model
-   ref
-   reactive
-   checkbox (boolean y array)
-   radio
-   select
-   modificadores
-   validación con computed
-   comunicación padre-hijo (emit + props)

# 🧠 Arquitectura

-   El estado principal vive en `App.vue`
-   `FormLibro.vue` emite el libro al padre
-   `ListaLibros.vue` recibe los libros por props
-   Single Source of Truth

# 🚀 Paso 1 --- App.vue

``` vue
<script setup>
import { ref } from 'vue'
import FormLibro from './components/FormLibro.vue'
import ListaLibros from './views/ListaLibros.vue'

const libros = ref([])

function agregarLibro(libro) {
  libros.value.push({
    ...libro,
    id: Date.now()
  })
}
</script>

<template>
  <div>
    <h1>📚 BookList SPA</h1>
    <FormLibro @agregar="agregarLibro" />
    <ListaLibros :libros="libros" />
  </div>
</template>
```

# 🚀 Paso 2 --- FormLibro.vue

``` vue
<script setup>
import { reactive, computed } from 'vue'

const emit = defineEmits(['agregar'])

const nuevoLibro = reactive({
  titulo: '',
  autor: '',
  descripcion: '',
  disponible: false,
  formatos: [],
  categoria: '',
  editorial: '',
  stock: 0
})

const formularioValido = computed(() => {
  return (
    nuevoLibro.titulo.length > 2 &&
    nuevoLibro.autor.length > 2 &&
    nuevoLibro.categoria !== '' &&
    nuevoLibro.editorial !== ''
  )
})

function enviarFormulario() {
  if (!formularioValido.value) return

  emit('agregar', { ...nuevoLibro })

  nuevoLibro.titulo = ''
  nuevoLibro.autor = ''
  nuevoLibro.descripcion = ''
  nuevoLibro.disponible = false
  nuevoLibro.formatos = []
  nuevoLibro.categoria = ''
  nuevoLibro.editorial = ''
  nuevoLibro.stock = 0
}
</script>
```

------------------------------------------------------------------------

## Template del Formulario

``` vue
<template>
  <form @submit.prevent="enviarFormulario">

    <input v-model.trim="nuevoLibro.titulo" placeholder="Título" />
    <input v-model.trim="nuevoLibro.autor" placeholder="Autor" />

    <textarea v-model="nuevoLibro.descripcion"></textarea>

    <label>
      <input type="checkbox" v-model="nuevoLibro.disponible" />
      Disponible
    </label>

    <h4>Formatos</h4>
    <label>
      <input type="checkbox" value="Físico" v-model="nuevoLibro.formatos" />
      Físico
    </label>
    <label>
      <input type="checkbox" value="Digital" v-model="nuevoLibro.formatos" />
      Digital
    </label>

    <h4>Categoría</h4>
    <label>
      <input type="radio" value="Novela" v-model="nuevoLibro.categoria" />
      Novela
    </label>
    <label>
      <input type="radio" value="Tecnología" v-model="nuevoLibro.categoria" />
      Tecnología
    </label>

    <select v-model="nuevoLibro.editorial">
      <option disabled value="">Seleccione editorial</option>
      <option>Planeta</option>
      <option>Penguin</option>
      <option>Nova</option>
    </select>

    <input type="number" v-model.number="nuevoLibro.stock" placeholder="Stock" />

    <button :disabled="!formularioValido">
      Guardar Libro
    </button>

  </form>
</template>
```

# 🚀 Paso 3 --- ListaLibros.vue

``` vue
<script setup>
defineProps({
  libros: Array
})
</script>

<template>
  <div>
    <h2>Lista de Libros</h2>

    <p v-if="libros.length === 0">
      No hay libros aún
    </p>

    <ul>
      <li v-for="libro in libros" :key="libro.id">
        <strong>{{ libro.titulo }}</strong>
        - {{ libro.autor }}
        ({{ libro.categoria }})
      </li>
    </ul>
  </div>
</template>
```
