# 🔟 Formularios dinámicos

📄 **Archivo:** `src/components/FormularioDinamico.vue`

```vue
<script setup>
import { reactive } from 'vue';

const formulario = reactive({});

const campos = [
  { id: 'nombre', label: 'Nombre', tipo: 'text', requerido: true },
  { id: 'email', label: 'Email', tipo: 'email', requerido: true },
  { id: 'edad', label: 'Edad', tipo: 'number', requerido: false },
  { id: 'comentarios', label: 'Comentarios', tipo: 'textarea', requerido: false },
];
</script>
```

Template:

```vue
<form>

<div v-for="campo in campos" :key="campo.id">

<label>{{ campo.label }}</label>

<input
v-if="campo.tipo !== 'textarea'"
v-model="formulario[campo.id]"
:type="campo.tipo"
>

<textarea
v-else
v-model="formulario[campo.id]"
></textarea>

</div>

</form>
```

# 11️⃣ Validación con objeto de errores

📄 **Archivo:** `src/components/ValidacionForm.vue`

```vue
<script setup>
import { reactive } from 'vue';

const usuario = reactive({
  nombre: '',
  email: '',
});

const errores = reactive({
  nombre: '',
  email: '',
});

function validarNombre() {
  if (!usuario.nombre) {
    errores.nombre = 'El nombre es requerido';
    return false;
  }

  errores.nombre = '';
  return true;
}
</script>
```

# 12️⃣ Componentes reutilizables con v-model

📄 **Archivo:** `src/components/CampoTexto.vue`

```vue
<script setup>
defineProps({
  modelValue: String,
  label: String,
  tipo: {
    type: String,
    default: 'text',
  },
});

defineEmits(['update:modelValue']);
</script>

<template>
  <label>{{ label }}</label>

  <input
    :value="modelValue"
    @input="$emit('update:modelValue', $event.target.value)"
    :type="tipo"
  />
</template>
```

Uso:

📄 **Archivo:** `src/components/FormularioContacto.vue`

```vue
<CampoTexto v-model="contacto.nombre" label="Nombre" />

<CampoTexto v-model="contacto.email" label="Email" tipo="email" />
```

Docs:

https://vuejs.org/guide/components/v-model.html

# 13️⃣ Estado global con Pinia

Instalar:

```bash
npm install pinia
```

📄 **Archivo:** `src/stores/encuestaStore.js`

```javascript
import { defineStore } from 'pinia';

export const useEncuestaStore = defineStore('encuesta', {
  state: () => ({
    datos: {},
  }),

  actions: {
    guardar(datos) {
      this.datos = datos;
    },
  },
});
```

Uso en componente:

📄 **Archivo:** `src/components/EncuestaForm.vue`

```javascript
import { useEncuestaStore } from '../stores/encuestaStore';

const store = useEncuestaStore();

function enviarEncuesta() {
  store.guardar(encuesta);
}
```

https://pinia.vuejs.org
